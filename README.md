# Chrome拡張 `Time tracker`

## インストール方法
1. 拡張機能ページを開く
	- `chrome://extensions/`
1. `デベロッパー モード`を有効にする
1. `パッケージ化されていない拡張機能を読み込む` から [./chrome-extension/](./chrome-extension) フォルダを選択する
1. 記録ページを開く
	- 拡張ボタンをクリックしてポップアップページを表示

## 使い方
1. （必要に応じて）分類を追加
1. （分類を細分化したい場合）ToDoを追加
1. 分類一覧またはToDo一覧の`開始`をクリック
1. 別の作業に切り替える場合は対応する`開始`をクリック
	- ※ 現在のものは自動で終了される
